Script Attacher

Do you want to attach Java scripts or CSS to a website?

With "Script Attacher" you can do this easily and permanently.
This add-on just takes your scripts and attaches them to the websites you can select with regular expressions.

Files

There are 4 types of files you can add:

  • JS
  • CSS
  • Script-link
  • Other

JS

This is a JavaScropt file.
It will be inlined in a script-tag

CSS

This is a CSS file.
It will be inlined in a style-tag

Script-link

This is a link to an external script.
It will be added as src in a script-tag

Other

This is JSON file.
It shoud have the keys "tagName" and "text".
"tagName" is the tagname of the added node (for example "link", "script" or "style").
"text" is the text appended as textnode.
All other keys will be added as attributes to the node (for example "src", "type", "href" or "rel").
